# d3charts

## Description

Chart library based on d3

## Install

1. Install dependencies

 ```
 npm install gulp-cli -g
 npm install
 ```

2. Build and start server

 ```
 gulp serve
 ```

The following page will open `http://localhost:9005/index.html
